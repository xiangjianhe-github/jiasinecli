# AI 对话界面优化文档

## 优化目标

将 JiasineCLI 的 AI 对话界面升级为类似 VS Code 和 Copilot CLI 的现代化风格，提供更美观的视觉体验和更好的 Markdown 渲染效果。

## 实施内容

### 1. ✅ Markdown 渲染增强

#### 新增特性
- **粗体支持**：`**文本**` → **粗体文本**
- **斜体支持**：`*文本*` 或 `_文本_` → *斜体文本*
- **改进的行内代码**：更鲜明的背景色和文字颜色对比
- **优化的代码块**：深灰色背景，更接近 IDE 体验

#### 技术实现
- 使用正则表达式处理顺序：行内代码 → 链接 → 粗体 → 斜体
- 避免格式冲突（如 `**` 不被解析为两个 `*`）
- 正确处理嵌套和边界情况

**文件修改：** `internal/render/markdown.go`

---

### 2. ✅ VS Code 暗色主题配色

#### 颜色方案升级
使用 ANSI 256 色调色板实现更丰富的颜色表现：

**基础颜色**（柔和，护眼）
- Cyan: `\033[38;5;80m` - 柔和青色
- Blue: `\033[38;5;75m` - 柔和蓝色
- Green: `\033[38;5;114m` - 柔和绿色
- Yellow: `\033[38;5;221m` - 柔和黄色
- Red: `\033[38;5;204m` - 柔和红色

**高亮颜色**（用于强调）
- BrightCyan: `\033[38;5;87m` - 亮青色（链接、标题）
- BrightBlue: `\033[38;5;111m` - 亮蓝色（关键词）
- BrightGreen: `\033[38;5;120m` - 亮绿色（成功、强调）
- BrightYellow: `\033[38;5;228m` - 亮黄色（警告）
- BrightRed: `\033[38;5;210m` - 亮红色（错误）

**背景色**（深色系）
- BgDarkGray: `\033[48;5;235m` - 深灰背景（代码块）
- ansiBgCode: `\033[48;5;236m` - 代码背景
- ansiCodeText: `\033[38;5;215m` - 行内代码文字（浅橙）

**灰色系**（次要信息）
- Gray: `\033[38;5;240m` - 深灰（注释、辅助）
- LightGray: `\033[38;5;250m` - 浅灰（次要内容）

**文件修改：** `internal/banner/banner.go`, `internal/render/markdown.go`

## 技术细节

### ANSI 转义序列
- `\033[1m` - 粗体
- `\033[3m` - 斜体
- `\033[4m` - 下划线
- `\033[38;5;Nm` - 256 色前景
- `\033[48;5;Nm` - 256 色背景
- `\033[0m` - 重置所有样式

### 正则表达式优化
- **粗体**：`\*\*([^*]+)\*\*`
- **斜体**：`\*([^*\s][^*]*?)\*`（避免匹配粗体）
- **行内代码**：`` `([^`]+)` ``
- **链接**：`\[([^\]]+)\]\(([^)]+)\)`

处理顺序确保：
1. 行内代码优先（避免内部被解析）
2. 链接次之（可能包含格式）
3. 粗体在斜体前（避免 `**` 被拆分）
4. 最后处理斜体
